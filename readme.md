# Setting up & Installing Dependencies

## Init Project
```bash
npm init -y
```

## Dev Dependencies

```bash
npm i -D typescript
npm i -D tslint
npm i -D ts-node
npm i -D nodemon
npm i -D @types/express
npm i -D @types/compression
```


## Project Dependencies

```bash
npm i -S express
npm i -S compression
```

## Configuring TypeScript

Setting up tsconfig.json

```bash
npm i -d typescript
```

Setting up tslint.json

```bash
./node_modules/.bin/tslint --init
```
